5G Core Network SDK Documentation

1. Introduction

Welcome to the 5G Core Network SDK (5gcsdk) documentation. This guide provides an overview of the SDK, its packages, usage, and setup requirements.

2. Packages

  • RFsimUEManager: Manage simulated UEs in the RF environment.
  • UEManager: Manage UEs within the 5G Core network.
  • CallbackManager: Register and handle event callbacks.
  • NFManager: Manage network functions within the 5G Core network.

Details for each package are available in the Namespaces section.

3. Description

The 5G Core Network SDK (5gcsdk) is a comprehensive development toolkit designed to facilitate the creation, testing, and management of 5G core network components. Tailored for network engineers and developers, this SDK provides a robust suite of tools and libraries to simulate, manage, and interact with various elements of a 5G core network environment.

4. Requirements

Ensure having python, pip, MongoDB installed and preferably with the following version :

  • Python 3.10.12
  • pip 22.0.2
  • MongoDB v6.0.8

4.1. Environment Setup

4.1.1. Using the setup.py file

You can install all required Python packages using the provided setup.py script in the 5gcsdk/src directory :

python3 setup.py install

4.1.2. Using the Provided Environment

You can use the virtual environment already set up in the repository (myenv) directly. Ensure to activate it using the provided command :

source myenv/bin/activate

5. Usage

  1. Import the oai5gc module.
  2. Initialize the OAI RFSIM5G environment.
  3. Use the functionalities provided by the SDK packages as needed.
  4. Terminate the session by calling the stop_handler() function.
